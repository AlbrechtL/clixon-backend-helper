Managed Switch CLI Overview

The managed switch CLI lets you interact with your network configuration using commands similar to Juniper or Cisco devices. You can view, edit, and commit configuration changes.

  • show – Display current configuration or state.
  • set – Modify configuration data.
  • delete – Remove configuration entries.
  • commit – Apply your changes.
  • quit – Leave the CLI session.
  • ? – Display help.

See the Clixon CLI Quickstart for more details.
